# Statistical-Computing-Project

This project analyzes credit card balances and their relationship with various demographic and financial factors. 
The dataset used for this project (Credit) can be found in the ISLR2 package in R. 

## Files
- StatCompProject.Rmd (see code)
- StatCompProject.html (see code and outputs)

## Skills and Methods
- Univariate and Multivariate Exploratory Data Analysis (EDA)
- Data visualization
- Correlation
- Effect size (including eta-squared and Cohen's d)
- Bootstrapping
- Analysis of outliers
